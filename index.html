Introdução ao Html

* (voltar)

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ela define a estrutura do conteúdo na web, permitindo que os navegadores interpretem e exibam o texto, imagens, vídeos e outros elementos.

Como é a estrutura básica

* (voltar)

Uma página HTML básica é composta por elementos que definem o cabeçalho, o corpo e outros componentes. Aqui está um exemplo simples:

     
<!DOCTYPE html>
<html lang="pt-br">   
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Bem-vindo ao HTML!</h1>
    <p>Este é um parágrafo de exemplo.</p>  
</body>
</html>

O que são atributos

* (voltar)

Os atributos fornecem informações adicionais sobre os elementos HTML. Eles são especificados dentro da tag de abertura do elemento.Exemplo:

<a href="https://www.dio.me/" target="_blank">Visite o Exemplo</a>

Neste exemplo, o atributo href define o link para o qual o elemento a aponta, e o atributo target="_blank" faz com que o link seja aberto em uma nova aba.

Links e Imagens

* (voltar)

Links são criados usando a tag a, enquanto imagens são inseridas com a tag img. Aqui está um exemplo de cada:

<a href="https://www.example.com">Clique aqui</a> <img src="imagem.jpg" alt="Descrição da Imagem">

O atributo src especifica o caminho da imagem, enquanto o atributo alt fornece uma descrição alternativa para a imagem.

O que são listas

* (voltar)

Listas podem ser ordenadas (ol) ou não ordenadas (ul).O exemplo de cada:

    
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
<ol>
    <li>Primeiro</li>
    <li>Segundo</li>       
    <li>Terceiro</li>
</ol>

O que são Tabelas

* (voltar)

Tabelas são criadas com a tag table, e cada linha é definida com tr, enquanto as células são definidas com td:

    
    
    <table>
    <tr>      
        
        <th>Coluna 1</th>
        <th>Coluna 2</th>  
                
    </tr>
    <tr>
        <td>Dados 1</td>
        <td>Dados 2</td>       
    </tr>
    <tr>  
        <td>Dados 3</td>
        <td>Dados 4</td>
    </tr>
    </table>

As tags th são usadas para cabeçalhos de tabela, enquanto td são usadas para células de dados.

O que é Formulário

* (voltar)

Formulários são usados para coletar dados do usuário. Eles são criados com a tag <form>, e os campos de entrada são definidos com tags como input, textarea e select.

Aqui está um exemplo simples:
    

    <form action="submit" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <br>
    <label for="mensagem">Mensagem:</label>
    <textarea id="mensagem" name="mensagem" rows="4" cols="50"></textarea>
    <br>
    <input type="submit" value="Enviar">
    </form>

O atributo action especifica para onde os dados do formulário serão enviados, enquanto o atributo method define o método HTTP a ser usado (geralmente "post" ou "get" ).
